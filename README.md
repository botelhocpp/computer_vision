# Computer Vision

Labs for the "Computer Vision" course.

## Labs List

There are a total of 9 labs:

1. Masking (in HSV) for Color Swap
2. Gamma Correction and Translation/Rotation 
3. Convolutional Filters
4. RANSAC
5. Seam Carving
6. Descriptors (SURF and ORB)
7. Homography
8. CNN (MNIST)
9. Face Recognition
